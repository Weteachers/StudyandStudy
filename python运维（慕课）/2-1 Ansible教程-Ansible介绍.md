- 了解 Ansible

# Ansible是什么
- Ansible是一个自动化管理IT资源的工具
  - 自动化
  - 管理
  - IT资源
# Ansible功能有哪些
- 系统环境配置
- 安装软件
- 持续集成
- 热回滚

# Ansible优点
- 无客户端
- 推送式
- 丰富的module
- 基于YAML的Playbook
- 商业化支持

# Ansible缺点
- 效率低、易挂起
- 并发性能差
