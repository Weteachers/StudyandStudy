# 编程模型

- 框架层面：actor, socket, map reduce...
- 应用：spark, hadooop, celery, vue...

# 函数式编程
- 把运算过程尽量写成一系列嵌套的函数调用
- 函数就是面向过程的程序设计的基本单元
- 功能地位与数据变量一样

# python中的函数式
- def：自定义函数
- lambda：匿名函数
- functools.partial：函数封装
- functools.wraps：装饰器
