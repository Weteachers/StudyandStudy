# Python中的基础数据结构
- 字符串：str, unicode
- 列表：list, tuple, set
  - list 内容可变 ——可变列表
  - tuple 内容不可变 ——不可变列表
  - set 里面的数据不可重复 ——集合【元素唯一性】
- 字典：dict
  - dict: key->value 映射，【元素唯一性】
