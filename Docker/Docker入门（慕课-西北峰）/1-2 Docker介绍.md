# 什么是Docker
- Docker: an open source project to **pack, ship and run** any appliaction as a lightweight container.
- Node.js: allows to package an application with all of its dependencies into a standardized unit.
- 类比
  - 可以粗糙地理解为轻量级的虚拟机
  - 开挂的chroot
- 确实不是虚拟机
